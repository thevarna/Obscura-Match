# Obscura Match — Frontend Application

This is the React frontend for Obscura Match, a confidential OTC crossing engine.

## Live Demo
[https://app-theta-ten-53.vercel.app/](https://app-theta-ten-53.vercel.app/)

## Quick Start
```bash
npm install
npm run dev
```

## Features
- **Glassmorphic UI**: Premium, institutional design system.
- **Real-time PER Connectivity**: Direct integration with MagicBlock Private Ephemeral Rollups.
- **Dynamic Auction Controls**: Admin panel for managing auction cycles and crank triggers.
- **Confidential Terminal**: Secure order submission and fund management.
- **Compliance Dashboard**: Transparent audit logs and permission management.

## Environment Variables
The app requires several `VITE_` prefixed variables to connect to the MagicBlock infrastructure. See `.env.development` for reference.
